Série de Comandos para Rodar no Terminal

Crie uma network no Docker para permitir a comunicação entre os containers:
docker network create cord_cont_network

Construa as imagens do front e do back:

docker build -t front-app -f Dockerfile.front .
docker build -t back-app -f Dockerfile.back .


Inicie o container do Back-end:

docker run -d --name back --network cord_cont_network -p 4000:4000 back-app

Inicie o container do Front-end:

docker run -d --name front --network cord_cont_network -p 3000:3000 front-app
